# 📊 Job Market Analyzer

A web-based application built using **Python, Streamlit, and SQL Server** that helps users analyze job market trends, salaries, in-demand skills, and hiring
companies through interactive dashboards.

---

##  Features

* Interactive dashboard
* Filter jobs by location, experience level, and job type
* View top hiring companies
* Analyze in-demand skills
* Salary Intelligence dashboard
* Company Insights
* Job Explorer
* Charts and KPI cards for better visualization

---

##  Tech Stack

* Python
* Streamlit
* SQL Server
* SQL
* Pandas
* PyODBC

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Job_Market_Analyzer.git
```

2. Navigate to the project folder

```bash
cd Job_Market_Analyzer
```

3. Install the required dependencies

```bash
pip install -r requirements.txt
```

4. Configure SQL Server and update the database connection.

5. Run the application

```bash
streamlit run app/streamlit_app.py
```
